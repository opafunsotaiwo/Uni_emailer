 🎓 UniBulk: University Bulk Email Sender

UniBulk is a beginner-friendly web application built with Flask and the SendGrid API. It allows university students, clubs, or departments to upload a CSV list of recipients and send personalized mass announcements without the risk of being flagged as spam by traditional email clients.

 🚀 Features
 CSV Upload: Seamlessly process recipient lists with name and email data.
 Personalization: Automatically addresses each student by name in the message body.
 API Integration: Uses SendGrid for high deliverability and professional tracking.
 Safety First: Includes a confirmation modal to review the message before "blasting."
 Results Dashboard: Provides immediate feedback on which emails were sent successfully.

 🛠️ Tech Stack
 Backend: Python / Flask
 Frontend: HTML5 / Bootstrap 5
 Email Engine: SendGrid SDK
 Security: Python-Dotenv for environment variable management
